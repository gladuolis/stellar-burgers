# Проектная работа "Stellar Burger"

Космическая бургерная — место, где любая форма жизни может заказать бургер своей мечты. Собери свой идеальный бургер из необычных ингредиентов, оформи заказ и следи за его приготовлением в реальном времени.

## Запуск

Для установки и запуска проекта выполните команды:

```bash
npm install
npm run start
```

или

```bash
yarn
yarn start
```

## Сборка

```bash
npm run build
```

или

```bash
yarn build
```

## Описание

Сервис по заказу космических бургеров с возможностью:

- Собирать бургер из различных ингредиентов
- Оформлять заказы (только для авторизованных пользователей)
- Просматривать ленту всех заказов
- Отслеживать историю своих заказов в личном кабинете
- Редактировать данные профиля

**Стек технологий:**

- React
- TypeScript
- Redux Toolkit
- React Router
- Webpack
- CSS Modules

**Структура проекта:**

```
src/
├── components/          # React компоненты
│   ├── app/             # Главный компонент приложения
│   ├── burger-constructor/  # Конструктор бургера
│   ├── burger-ingredients/  # Список ингредиентов
│   ├── order-card/      # Карточка заказа
│   └── ...
├── pages/               # Компоненты страниц
├── services/            # Redux store и слайсы
│   ├── slices/          # Слайсы Redux
│   └── store.ts         # Конфигурация store
├── utils/               # Вспомогательные функции
│   ├── burger-api.ts    # API запросы
│   ├── cookie.ts        # Работа с куками
│   └── types.ts         # TypeScript типы
└── index.tsx            # Точка входа
```
