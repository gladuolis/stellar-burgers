# Проектная работа "Stellar Burger"

Космическая бургерная — место, где любая форма жизни может заказать бургер своей мечты. Собери свой идеальный бургер из необычных ингредиентов, оформи заказ и следи за его приготовлением в реальном времени.

## Запуск

Для установки и запуска проекта выполните команды:

```bash
npm install
npm run start
```
или

```bash
yarn
yarn start
```
## Сборка

```bash
npm run build
```
или

```bash
yarn build
```

## Тестирование

Проект покрыт тестами с использованием Cypress и Jest.

**Запуск Cypress тестов:**

```bash
npm run cypress:open   # интерактивный режим
npm run cypress:run    # headless режим
```

**Запуск Jest тестов:**

```bash
npm test               # однократный запуск
npm run test:watch     # режим наблюдения
```
**Cypress тесты проверяют:**

Загрузку и отображение ингредиентов

Добавление булки и начинки в конструктор

Открытие страницы ингредиента

Возврат на главную по кнопке "Назад"

Закрытие модального окна (оверлей и крестик)

Создание заказа авторизованным пользователем

Очистку конструктора после успешного заказа

**Jest тесты проверяют:**

Корректную работу rootReducer

Редьюсеры конструктора (добавление, удаление, перемещение, очистка)

Редьюсеры ингредиентов (pending, fulfilled, rejected)

Редьюсеры заказов (pending, fulfilled, rejected, очистка)

Редьюсеры ленты заказов

Редьюсеры истории заказов

Редьюсеры пользователя

## Описание

Сервис по заказу космических бургеров с возможностью:

Собирать бургер из различных ингредиентов

Оформлять заказы (только для авторизованных пользователей)

Просматривать ленту всех заказов

Отслеживать историю своих заказов в личном кабинете

Редактировать данные профиля

**Стек технологий:**

React

TypeScript

Redux Toolkit

React Router

Webpack

CSS Modules

Cypress (e2e тестирование)

Jest (unit тестирование)
